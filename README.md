# Netflix Data Exploratory Data Analysis (EDA)

## Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on a **real-world Netflix Dataset**. The primary objective is to analyze actual movie and TV show data to uncover hidden trends, content strategies, and patterns within one of the world's largest streaming platforms. This project practically demonstrates the core concepts of data cleaning, manipulation, and visualization using industry-standard data.

## Key Features & Analysis Performed
* **Data Cleaning & Preprocessing**: Handled missing values, corrected column data types, and removed duplicate records to ensure an accurate analysis of the Netflix catalog.
* **Genre Analysis**: Identified the most frequently produced and highest-performing genres on the platform.
* **Rating & Engagement Trends**: Analyzed the distribution of audience ratings and total votes to understand content popularity and user engagement trends.
* **Feature Correlation**: Explored correlations between release years, content types, and audience ratings to find patterns in content success.
* **Visualizations**: Utilized Bar Plots, Histograms, and Heatmaps to effectively present data-driven insights and business trends.

## Technologies & Libraries Used
* **Language**: Python
* **Data Manipulation**: Pandas, NumPy
* **Data Visualization**: Matplotlib, Seaborn
* **Environment**: Jupyter Notebook / VS Code

## Key Insights Discovered
* **Dominant Genres**: Specific genres (such as Drama, Comedy, and Action) dominate the Netflix dataset in terms of overall production volume.
* **Rating Impact**: A strong positive correlation was observed between higher audience ratings and total votes, indicating high user engagement for critically acclaimed content.
* **Data Distribution**: The analysis revealed that the majority of Netflix content falls within a specific rating bracket, providing a clear picture of general audience reception over the years.
